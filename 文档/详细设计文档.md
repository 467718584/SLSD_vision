## 3. 数据流设计

### 3.1 数据集上传流程

```
1. 用户选择上传模式(ZIP/文件夹)
2. 填写数据集名称、算法类型等
3. 点击上传
4. 后端接收文件
5. 解压/保存到目录
6. 统计图片和标注数量
7. 校验标注格式(可选)
8. 生成元数据
9. 保存到数据库
10. 返回结果，前端刷新
```

### 3.2 模型上传流程

```
1. 用户选择模型文件夹
2. 填写模型信息
3. 点击上传
4. 后端接收文件
5. 保存到模型目录
6. 解析results.csv获取精度
7. 生成训练曲线图
8. 生成元数据
9. 保存到数据库
10. 返回结果，前端刷新
```

### 3.3 精度曲线展示流程

```
1. 用户点击模型列表精度曲线缩略图
2. 跳转到模型详情页面
3. 调用 /api/model/detail/<name>
4. 后端检查曲线图是否存在
5. 如不存在，生成曲线图
6. 返回图表路径
7. 前端渲染图表
```
