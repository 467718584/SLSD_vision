# 机器视觉管理平台需求分析文档

## 1. 引言

### 1.1 文档目的

本文档旨在详细描述机器视觉管理平台的功能需求和非功能需求，为开发团队提供明确的设计依据，同时也为测试团队提供验收标准。

### 1.2 项目背景

机器视觉管理平台（SLSD Vision Platform）是一个用于管理和展示机器视觉相关数据集与训练模型的Web平台。项目旨在解决实际应用中数据集和模型文件分散管理、缺乏可视化预览等问题。

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| 数据集(Dataset) | 包含训练图片和标注文件的数据集合 |
| 模型(Model) | 训练好的机器视觉模型文件 |
| YOLO格式 | 一种常用的目标检测标注格式 |
| VOC格式 | Pascal VOC数据集使用的标注格式 |
| COCO格式 | MS COCO数据集使用的标注格式 |
| mAP | mean Average Precision，平均精度均值 |
| 应用现场 | 模型的部署应用场景/数据来源 |

## 2. 功能需求

### 2.1 数据集管理

#### 2.1.1 数据集上传

**功能描述**：支持用户上传数据集到系统

**需求详情**：
- 支持ZIP压缩包方式上传
- 支持文件夹（webkitdirectory）方式上传
- 支持最大5GB的文件上传
- 上传前进行名称重复检测
- 支持选择标注类型（YOLO/VOC/COCO）
- 可选的YOLO格式4步校验

**优先级**：高

#### 2.1.2 数据集列表展示

**功能描述**：在页面上展示所有数据集

**需求详情**：
- 显示算法类型标签（颜色区分）
- 显示样本数量和标签数量
- 显示维护日期和人员
- 显示数据集缩略图预览
- 支持按算法类型筛选
- 支持按名称搜索

**优先级**：高

#### 2.1.3 数据集详情查看

**功能描述**：查看数据集的详细信息

**需求详情**：
- 显示数据集基本信息（名称、描述、算法类型等）
- 显示样本分布图表（train/val/test）
- 显示类别分布热力图
- 显示标注可视化预览
- 支持图片点击放大预览

**优先级**：高

#### 2.1.4 数据集编辑

**功能描述**：编辑数据集属性

**需求详情**：
- 支持修改算法类型
- 支持修改技术方法
- 支持上传/更换样本分布图
- 支持上传/更换类别分布图
- 支持编辑标签（类别名称）

**优先级**：中

#### 2.1.5 数据集删除

**功能描述**：删除数据集

**需求详情**：
- 删除前进行确认提示
- 同步删除文件系统中的数据
- 同步删除数据库记录

**优先级**：中

### 2.2 模型管理

#### 2.2.1 模型上传

**功能描述**：上传训练好的模型

**需求详情**：
- 支持文件夹方式上传
- 自动解析results.csv获取模型精度（mAP50）
- 自动生成训练曲线图表
- 支持选择算法类型、技术方法、模型类别
- 支持填写模型描述

**优先级**：高

#### 2.2.2 模型列表展示

**功能描述**：在页面上展示所有模型

**需求详情**：
- 显示编号、算法类型、技术方法
- 显示模型名称、类别、概述
- 显示精度进度条（颜色区分）
- 显示应用现场
- 显示关联数据集
- 显示精度曲线缩略图（4张）
- 支持按算法类型筛选
- 支持按名称搜索

**优先级**：高

#### 2.2.3 模型详情查看

**功能描述**：查看模型的详细信息

**需求详情**：
- 显示模型基本信息
- 显示精度详情
- 显示模型详细精度曲线（mAP50、mAP50-95、损失曲线）
- 显示PR曲线与混淆矩阵
- 显示验证集识别结果
- 支持图片点击放大预览

**优先级**：高

#### 2.2.4 模型删除

**功能描述**：删除模型

**需求详情**：
- 删除前进行确认提示
- 同步删除文件系统中的数据
- 同步删除数据库记录

**优先级**：中

### 2.3 应用现场管理

#### 2.3.1 应用现场列表

**功能描述**：展示所有应用现场

**需求详情**：
- 根据设置页面的应用现场词条自动生成
- 显示每个现场关联的数据集数量
- 显示每个现场关联的模型数量
- 显示关联的数据集名称
- 显示关联的模型名称

**优先级**：中

### 2.4 系统设置

#### 2.4.1 设置管理

**功能描述**：管理系统配置词条

**需求详情**：
- 管理算法类型（应用场景）词条
- 管理技术方法词条
- 管理标注格式词条
- 管理应用现场词条
- 支持新增和删除词条
- 设置持久化保存到数据库

**优先级**：中

### 2.5 原始数据管理

#### 2.5.1 原始数据上传

**功能描述**：上传原始数据条目

**需求详情**：
- 支持填写数据名称、算法类型
- 支持选择数据来源（应用现场）
- 支持填写关联数据集
- 支持标记是否已标注

**优先级**：低

#### 2.5.2 原始数据列表

**功能描述**：展示所有原始数据

**需求详情**：
- 显示编号、名称、算法类型
- 显示数据来源、是否已标注
- 显示维护日期和人员

**优先级**：低

### 2.6 统计概览

#### 2.6.1 全体总览

**功能描述**：展示平台整体统计信息

**需求详情**：
- 显示数据集总数、模型总数
- 显示各算法类型分布

**优先级**：中

## 3. 非功能需求

### 3.1 性能需求

| 指标 | 要求 |
|------|------|
| 页面加载时间 | < 3秒 |
| API响应时间 | < 1秒 |
| 大文件上传 | 支持最大5GB |
| 并发用户 | 单用户 |

### 3.2 兼容性需求

| 浏览器 | 版本要求 |
|--------|----------|
| Chrome | 最新版 |
| Edge | 最新版 |
| Firefox | 最新版 |

### 3.3 可靠性需求

- 数据持久化存储
- 上传进度实时显示
- 错误提示清晰明确

### 3.4 可维护性需求

- 模块化代码结构
- 配置文件与代码分离
- 完整的错误日志

### 3.5 安全性需求

- 输入数据转义处理
- 文件名安全校验
- 路径遍历防护

## 5. 接口需求

### 5.1 REST API

| 模块 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 数据集 | GET | /api/datasets | 获取数据集列表 |
| 数据集 | POST | /api/dataset/upload | 上传数据集 |
| 数据集 | DELETE | /api/dataset/<name> | 删除数据集 |
| 模型 | GET | /api/models | 获取模型列表 |
| 模型 | POST | /api/model/upload | 上传模型 |
| 模型 | GET | /api/model/detail/<name> | 获取模型详情 |
| 模型 | DELETE | /api/model/<name> | 删除模型 |
| 设置 | GET | /api/settings | 获取系统设置 |
| 设置 | POST | /api/settings | 更新系统设置 |
| 统计 | GET | /api/stats | 获取统计信息 |

## 6. 验收标准

### 6.1 功能验收

| 功能点 | 验收标准 |
|--------|----------|
| 数据集上传 | 能成功上传ZIP/文件夹，数据存储正确 |
| 数据集预览 | 能显示图片和标注，可点击放大 |
| 数据集校验 | YOLO格式校验功能正常 |
| 标签编辑 | 能修改类别名称并保存 |
| 模型上传 | 能上传文件夹并解析精度 |
| 曲线生成 | 自动生成4种曲线图 |
| 应用现场 | 根据设置词条自动生成列表 |
| 系统设置 | 能新增/删除词条并持久化 |

### 6.2 界面验收

| 验收点 | 验收标准 |
|--------|----------|
| 页面布局 | 清晰美观，响应式设计 |
| 颜色区分 | 算法类型、精度等级颜色正确 |
| 图表显示 | 图表正常渲染，中文显示正确 |
| 交互反馈 | 按钮点击、弹窗显示正常 |
