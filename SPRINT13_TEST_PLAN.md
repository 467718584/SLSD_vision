## 三、测试执行方法

### 工具选择
使用 Playwright 进行浏览器自动化测试

### 安装Playwright
```bash
npm install -g playwright
npx playwright install chromium
```

### 截图保存规则
```
screenshots/
├── login/
│   ├── 1.1_login_page.png
│   ├── 1.2_login_success.png
│   └── 1.3_login_error.png
├── datasets/
│   ├── 2.1_dataset_list.png
│   └── ...
├── models/
├── overview/
├── settings/
└── ui/
```

### 执行命令
```bash
# 启动测试
npx playwright test --project=chromium --reporter=list

# 或使用脚本
./run_ui_test.sh
```
