## Roboflow 风格特点

| 元素 | Roboflow 风格 |
|------|--------------|
| 主色 | #1E293B (深灰蓝) |
| 强调色 | #3B82F6 (蓝色) |
| 成功色 | #10B981 (绿色) |
| 背景 | #0F172A (深蓝黑) |
| 卡片 | #1E293B (略浅) |
| 文字 | #F8FAFC (白) |
| 次要文字 | #94A3B8 (灰) |
| 边框 | #334155 (暗灰) |
